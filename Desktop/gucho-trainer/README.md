# 具抽トレーナー

具体と抽象を往復する力を鍛えるためのトレーニングアプリです。

## 目的

面接や対話の中で、自分の経験を具体的に語る力と、経験から本質・構造・学びを抽象化する力を鍛えます。

## 主な機能

- 面接練習
  - 入力したテーマについて、面接官が具体化・抽象化の観点で深掘りします。
- 抽象化ドリル
  - 対になる言葉を上位概念で捉え、「どういう意味で一緒か」を表現する練習をします。
- 振り返り
  - 会話内容から、具体と抽象のバランスや往復力を確認します。

## 技術スタック

- React
- TypeScript
- Vite
- Tailwind CSS
- Anthropic API

## 開発コマンド

```sh
npm install
npm run dev
npm run build
```
